作品说明

📱 项目简介

这是一个基于美团单车的游戏化骑行记录与激励工具的前端高保真原型，旨在通过游戏化机制提升用户骑行体验和粘性。

💡 构建思路

本项目采用 Vibe Coding 方式，借助大模型（Claude Opus 4.6）完成从需求分析到原型落地的全流程：

  • 需求与竞品分析：对哈啰出行、青桔单车、美团现状进行深度分析，发现美团在游戏化激励方面完全空白，而其生态优势（外卖、酒旅联动）是独特护城河
  • 核心设计方向：构建统一的游戏化世界观，以"城市探索者"为核心身份，将骑行数据转化为游戏进度，让每次出行都有意义
  • 界面设计文档：依托大模型撰写了详细的界面构建规范，明确了16个页面的布局、交互流程、视觉规范和动效方案
  • 原型实现：基于设计文档，通过 Vibe Coding 迭代式构建，最终产出可交互的高保真原型

🏗️ 系统架构（7层体系）

整个游戏化系统围绕"探索-成长-分享"三大核心展开，由7个功能层组成：

  • 基础层 — 骑行记录：自动记录骑行数据，提供可视化统计（ECharts图表）
  • 核心层 — 探索者成长：1-100级等级体系 + 双货币系统（能量⚡养成 / 星币⭐兑换）+ 技能树
  • 养成层 — 伙伴与单车：虚拟伙伴陪伴骑行、解锁故事章节；虚拟单车升级、提升属性加成
  • 探索层 — 城市探索：区域解锁、特色路线发现、城市记忆收集
  • 挑战层 — 挑战赛：季度主题挑战、城市PK、每日任务、勋章排行
  • 社区层 — 骑行社区：路线分享、打卡动态、好友互动
  • 生态层 — 美团联动：星币兑换美团神券、会员成长加速、路线商家推荐

✨ 核心特性

  • 双货币系统：能量⚡用于养成，星币⭐用于兑换美团优惠，互不混淆
  • 探索者成长：1-100级等级体系，解锁新功能和权益
  • 伙伴养成：虚拟伙伴陪伴骑行，喂养成长、解锁技能与故事
  • 单车养成：虚拟单车升级，提升能量/经验/星币加成
  • 技能树系统：多分支技能解锁，提供各类骑行加成
  • 挑战系统：季节性挑战、每日任务、城市PK
  • 城市探索：区域解锁、路线收集、城市记忆
  • 社区互动：路线分享、打卡动态、点赞评论

🎨 设计亮点

  • 美团品牌色调，游戏化视觉风格
  • 16个页面完整覆盖（5个主页面 + 11个详情页）
  • ECharts数据可视化集成
  • 丰富的动画效果（进度条、数字增长、呼吸浮动等）
  • 移动端优先的响应式设计

🛠️ 技术实现

  • 技术栈：HTML5 + CSS3 + JavaScript（原生，无框架）
  • 架构：Hash路由单页应用，模块化设计
  • 数据：完整的Mock数据体系
  • 代码量：4,500+ 行代码，43个文件

🎯 使用说明

本原型为静态展示版本，所有数据均为模拟数据。您可以：

  • 点击底部5个Tab切换主页面
  • 点击首页各元素进入详情页
  • 浏览图鉴、查看数据统计
  • 体验完整的游戏化交互流程
