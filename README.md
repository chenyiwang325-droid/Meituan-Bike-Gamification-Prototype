# 骑行能量站 - 美团单车游戏化原型

这是一个基于美团单车的游戏化骑行记录与激励工具的前端高保真原型。

## 功能特性

### 核心系统
- **双货币系统**：能量（⚡）用于养成，星币（⭐）用于兑换美团优惠
- **探索者等级**：1-100级成长体系，解锁新功能和权益
- **伙伴养成**：虚拟伙伴陪伴骑行，提供加成和故事
- **单车养成**：虚拟单车升级，提升属性加成
- **技能树**：解锁各类加成技能
- **挑战系统**：季节性挑战、每日任务
- **城市探索**：区域解锁、路线收集
- **社区互动**：路线分享、打卡动态

### 页面结构

#### 主页面（5个Tab）
1. **首页** - 骑行数据仪表盘、游戏化元素展示
2. **探索** - 城市区域、特色路线推荐
3. **骑行** - 扫码骑行入口
4. **社区** - 动态分享、路线推荐
5. **我的** - 个人中心、图鉴、兑换

#### 详情页面（11个）
- 探索者详情页
- 伙伴与单车详情页
- 技能树页面
- 我的挑战页面
- 数据统计页面
- 单次骑行详情页
- 路线详情页
- 勋章图鉴页
- 单车图鉴页
- 伙伴图鉴页
- 星币兑换页

## 技术栈

- **HTML5 + CSS3 + JavaScript**（原生，无框架）
- **ECharts 5.4.3**（数据可视化）
- **Hash路由**（单页应用）
- **响应式设计**（移动端优先，375px基准）

## 运行方式

### 方法1：直接打开
```bash
# 在浏览器中直接打开
open prototype/index.html
# 或双击 index.html 文件
```

### 方法2：使用本地服务器
```bash
# 使用 Python
cd prototype
python -m http.server 8000

# 使用 Node.js
npx serve prototype

# 使用 PHP
cd prototype
php -S localhost:8000
```

然后在浏览器访问 `http://localhost:8000`

## 设计规范

### 颜色系统
- 主色：美团黄 `#FFD100`
- 能量金：`#FFC300`
- 星币金：`#FFD700`
- 经验蓝：`#4A90E2`
- 史诗紫：`#9C27B0`
- 传奇橙：`#FF6B00`
- 成功绿：`#52C41A`

### 组件规范
- 卡片圆角：16-20px
- 阴影：`0 4px 16px rgba(0,0,0,0.12)`
- 进度条高度：8-12px
- 移动端宽度：375px基准

## 数据说明

所有数据均为Mock数据，存储在 `js/store.js` 中。包括：
- 用户信息（等级、经验、货币）
- 骑行记录
- 伙伴和单车数据
- 技能、挑战、勋章
- 路线和区域信息
- 社区动态

## 动画效果

- 页面切换淡入动画
- 进度条填充动画

- 伙伴呼吸浮动

## 开发说明

本原型为静态展示版本，不包含：
- 后端API
- 真实地图服务
- 用户认证
- 数据持久化

所有交互和数据均在前端模拟实现。
